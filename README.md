# 🛒 Bot de Gestión de Almacén

Sistema de gestión para Telegram conectado con **Supabase**.

## 🚀 Funciones actuales
* **Carrito de compras**: Suma de productos en tiempo real.
* **Tickets de venta**: Generación de links para WhatsApp (Efectivo, Transferencia, Fiado).
* **Reportes PDF**: Generación de informes de ventas diarias.
* **Caja del día**: Consulta rápida de ingresos.

## 🛠️ Instalación
1. Clonar el repositorio.
2. Instalar dependencias: `pip install -r requirements.txt`.
3. Configurar las variables en el archivo `.env`.
4. Ejecutar: `python bot.py`.
